# 卡拉彼丘喵言喵语：
该插件可以发送卡拉彼丘的喵语。一起成为卡奴吧喵！

  <img src="image/CalabiYau.png" alt="卡拉彼丘" width="200">

## 食用方法：
  1. 下载 `喵言喵语.js` 插件到云崽Bot的 `plugins/example` 目录下。
  2. 发送`喵言喵语`即可。

脚本会自动初始化词库文件 `CalabiYau_text.yml`。

## 其他：
  - 插件默认使用的是Gitee源。
  - 由于国内审查机制，直接明文存放词库仓库会炸，故使用base64编码存放。
  - 编码后的文件为 `CalabiYau_text_base64.txt`
